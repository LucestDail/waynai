  "keyword": "키워드 (intent가 keyword 또는 area_keyword인 경우)",
  "area": {
    "name": "지역명",
    "code": "지역코드",
    "sigungu": {
      "name": "시군구명",

    }
  },
  "confidence": 0.95
}

1. keyword: "맛집", "카페", "박물관", "해변", "산", "쇼핑" 등 관광 키워드
2. area: "서울", "부산", "제주도", "강릉", "경주" 등 지역명
